PhobOS delivers all the security and ease of use of Debian, without the complexity of Yocto or Containers. PhobOS was designed to be compatible with secure atomic updates using [torizon.io](https://www.torizon.io/) platform. Do not update only Containers, update the whole system with a single atomic update.
